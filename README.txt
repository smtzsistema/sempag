PARA UMA INSTALAÇÃO LIMPA FAÇA O DOWNLOAD DOS ARQUIVOS SEM A PASTA vendor ASSIM TEMOS MAIS CERTEZA DE QUE TUDO FUNCIONARÁ PERFEITAMENTE NA SUA MAQUINA!!!

RODE OS COMANDOS NO POWERSHELL PARA UMA INSTALAÇÃO LIMPA:


CLONA O REPOSITORIO
cd C:\laragon\www
git clone SEU_REPO sempag
cd sempag

CRIA UMA COPIA DO .ENV
copy .env.example .env

INSTALA AS BIBLIOTECAS (PASTA vendor)
composer install

GERA A CHAVE DE ACESSO DO SISTEMA PARA CRIPTOGRAFIAS E CONFIGURAÇÕES DO LARAVEL
php artisan key:generate

CRIA A ESTRUTURA DE BANCO DE DADOS
php artisan migrate --force

CRIA TODAS AS LINHAS NECESSARIAS PARA O SISTEMA FUNCIONAR E TODOS OS ARQUIVOS INICIAIS
php artisan db:seed --force

///////////////////////////

UTILIZANDO O LARAGON BASTA INICAR O APACHE E O MYSQL E NO POWER SHELL RODAR:

php artisan serve
